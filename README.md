# Stratagem v2

> "不是教你耍狠，是帮你在复杂局面里，看清棋盘，找到那一步。"

一个面向 AI Agent 的**策略分析技能包 (AgentSkill)**，用于谈判、竞争、定位和复杂人际博弈的深度局势判断。

---

## 它解决什么问题

你是否遇到过这种情况：

- 一件事表面上在吵，真正的矛盾却不在台面上
- 对方看起来在谈合作，本质上却在试探你的底线
- 你知道不能硬碰硬，但又不知道该绕到哪里去赢
- 你不缺道理，缺的是**看清局势后的那一步**

Stratagem 就是为这种局面准备的。

它不给你一堆泛泛建议。它会帮你：

1. **看清局势表层之下真正的结构**
2. **识别每个人在"演什么戏"vs 真实意图**
3. **找到核心矛盾和 leverage point**
4. **从多个策略传统里抽出一个主计**
5. **给你可执行的话术、动作和时机**
6. **推演对方的反制，准备好第二手**

---

## 融合的策略体系

v2 融合了五套不同风格的策略框架：

| 体系 | 核心价值 |
|------|----------|
| **孙子兵法** | 势、虚实、谋攻、知彼知己、不战而屈人之兵 |
| **三十六计** | 借势、诱导、转移、反转、脱困的具名招式 |
| **君主论** | 权力维持、名望管理、恐惧 vs 爱戴、政治现实主义 |
| **毛选（节选思路）** | 矛盾分析法、阶段判断、持久战、以弱胜强 |
| **江学勤博弈框架** | 决策主体拆解、成本/升级阶梯分析、叙事与景观政治 |

### 江学勤三要素

v2 最大的新增是将江学勤 (Jiang Xueqin) 的"预测性历史"博弈分析方法融入输出结构：

1. **决策主体映射 (Actor Mapping)** — 列出所有博弈参与者的核心利益、约束条件和决策逻辑
2. **博弈逻辑与成本计算 (Game Logic & Cost)** — 零和 vs 正和判断、廉价胜利陷阱、持久战成本、升级阶梯分析
3. **叙事与景观政治 (Narrative & Optics)** — 识别各方的"受众"、表演行为与真实意图、叙事劫持机会

---

## 输出结构（9 段）

1. **局势判断** — 表面 vs 实质
2. **决策主体映射** — 谁在场、谁在幕后、各方真实利益
3. **博弈逻辑与成本** — 博弈结构、廉价胜利诱惑、时间站谁这边
4. **叙事与景观政治** — 谁在演戏、演给谁看、能否劫持叙事
5. **核心矛盾** — 主要矛盾、矛盾主要方面、动态转化
6. **主计** — 只给一个最值得执行的策略
7. **为什么是这一计** — 匹配度论证
8. **行动剧本** — 话术、动作序列、时机节点
9. **博弈推演与风险** — 对方反制、二手准备、失败信号、止损线

## 适合什么场景

- 谈判僵局怎么破
- 对方到底在打什么算盘
- 职场博弈里怎么拿主动权
- 竞争对手在释放什么信号
- 一个复杂局面，先打哪一点最有效
- 要不要硬碰、要不要拖、要不要绕、要不要借力

一句话：当问题已经不是"怎么做事"，而是**"怎么在复杂关系和约束里推进事情"**时，它就有用。

---

## 它不会做什么

这个 skill 不是给暴力、骚扰、欺骗、犯罪规避之类场景准备的。

它适合：普通竞争、合法谈判、冲突管理、组织策略、关系判断、舆论与定位分析。

边界很简单：**可以讲策略，不帮你越线。**

## 这个 skill 的核心气质

不是热血。不是鸡汤。也不是"教你拿捏别人"。

它更像一种冷静的能力：
- 先把棋盘看清
- 再识别矛盾
- 看穿谁在演戏
- 最后只走那一步最值钱的棋

如果你喜欢的不是空洞建议，而是**局势感、节奏感、分寸感**，那它就是为你准备的。
