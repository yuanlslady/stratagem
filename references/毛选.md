# 毛泽东选集

> 毛泽东思想核心 | 哲学·军事·政治·方法论的完整体系

## 一、哲学方法类

### 实践论（1937年）

**写作背景**：延安时期，针对党内教条主义和经验主义错误而作，是马克思主义认识论的中国化系统表述。

**核心观点**：

1. **认识的起点——实践**
   - 人的认识依赖于社会实践，脱离实践的认识是空洞的
   - 实践包括：生产活动、阶级斗争、政治生活、科学和艺术活动
   - "你要有知识，你就得参加变革现实的实践"

2. **认识的两个阶段**
   - **感性认识**：直接接触事物，获得印象和表象（现象层）
   - **理性认识**：通过思维加工，抓住本质、规律和内在联系（本质层）
   - 从感性到理性是认识的第一次飞跃——需要足够的感性材料和科学的思维加工

3. **认识的第二次飞跃——理论回归实践**
   - 更重要的飞跃：从理性认识到革命实践
   - 理论如不能指导行动，就没有价值；计划如不经实践检验，就不知真假
   - "认识从实践始，经过实践得到了理论的认识，还须再回到实践去"

4. **认识的螺旋式发展**
   - 实践 → 认识 → 再实践 → 再认识（无限循环，螺旋上升）
   - 每次循环都在更高层次上加深对事物的认识
   - 真理是相对的，人类认识随实践发展而不断深化

5. **真理标准**
   - 实践是检验真理的唯一标准
   - 客观现实变化无穷，认识也永远没有终点
   - 反对"本本主义"——不能拿理论当圣经，一切要通过实践检验

**完整原文名句**：
- "通过实践而发现真理，又通过实践而证实真理和发展真理"
- "你要知道梨子的滋味，你就得变革梨子，亲口吃一吃"
- "马克思主义者认为，只有人们的社会实践，才是人们对于外界认识的真理性的标准"
- "没有调查，没有发言权"

**核心逻辑链**：
```
感性实践 → 感性认识 → 理性认识 → 指导实践 → 检验/深化 → 新的感性实践
```

**现代应用**：
- 投研：一手调研优于二手数据，实地调查优于数据库报告
- 决策：任何假设都要设计"可证伪"的检验方案
- 学习：读书、做笔记不算学会，用于解决真实问题才算学会
- 反教条：警惕"XX方法论说……"作为唯一论据，必须落地验证

### 矛盾论（1937年）

**写作背景**：与《实践论》同期，系统阐述唯物辩证法，批判教条主义对马克思主义的僵化理解。

**核心观点**：

1. **矛盾的普遍性（矛盾无处不在）**
   - 矛盾存在于一切事物的发展过程中
   - 每一事物的内部自始至终都存在矛盾运动
   - 没有矛盾就没有世界——这是辩证法的根本命题
   - "一万年都有两点"

2. **矛盾的特殊性（具体问题具体分析）**
   - 不同事物有不同的矛盾，同一事物在不同阶段有不同的矛盾
   - **不同质的矛盾只能用不同质的方法解决**
   - 中国革命的矛盾不同于俄国革命——不能照搬苏联模式
   - 研究矛盾须分析其特殊性，切忌一刀切

3. **主要矛盾与次要矛盾**
   - 复杂事物中，必有一种矛盾处于支配地位，是**主要矛盾**
   - 主要矛盾决定事物发展的方向和进程
   - **"捉住了主要矛盾，一切问题就迎刃而解了"**
   - 次要矛盾不可忽视，特定条件下主次可以转化

4. **矛盾的主要方面与次要方面**
   - 一个矛盾的两个方面，必有一方处于支配地位（主要方面）
   - 事物的性质由矛盾的主要方面决定
   - 矛盾两方面的地位可以相互转化（量变引起质变）

5. **矛盾的同一性与斗争性**
   - **同一性**：矛盾双方相互依存，互为条件，在一定条件下可以相互转化
   - **斗争性**：矛盾双方相互排斥，相互对立，这是绝对的
   - 有条件的同一性 + 无条件的斗争性 = 矛盾运动的全部真相
   - 斗争性推动事物发展，同一性规定转化方向

6. **对抗性矛盾与非对抗性矛盾**
   - 对抗性矛盾：双方根本利益冲突，无法调和（敌我矛盾）
   - 非对抗性矛盾：双方基本利益一致，可以调和（人民内部矛盾）
   - 两类矛盾不是一成不变的，处理不当可以相互转化

**矛盾分析四步法（实操框架）**：
```
① 识别矛盾 → 这件事的核心张力在哪里？
② 找主要矛盾 → 哪个矛盾决定全局？
③ 分析特殊性 → 这个具体矛盾有什么独特性？
④ 确定方法 → 对抗性还是非对抗性？用什么手段解决？
```

**完整原文名句**：
- "事物的矛盾法则，即对立统一的法则，是唯物辩证法的最根本的法则"
- "在复杂的事物的发展过程中，有许多的矛盾存在，其中必有一种是主要的矛盾"
- "不同质的矛盾，只能用不同质的方法才能解决"
- "研究问题，忌带主观性、片面性、表面性"
- "任何运动形式，其内部都包含着本身特殊的矛盾"

**现代应用**：
- 投研：任何公司都有核心矛盾（增长 vs 利润率、扩张 vs 护城河），找准才能判断拐点
- 组织：区分团队内部的"可调和矛盾"（流程效率问题）与"原则性矛盾"（价值观冲突）
- 决策：不要用同一套框架分析所有问题，先识别这个问题的特殊性
- 战略：永远问"当前主要矛盾是什么"，而不是试图同时解决所有问题

### 反对本本主义（1930年）

**写作背景**：红军时期，针对部分同志生搬硬套苏联经验、脱离中国实际的"本本主义"倾向。

**核心观点**：

1. **调查是一切工作的前提**
   - 没有调查，就没有发言权
   - 调查就是解决问题——不是先有结论再去找证据
   - "调查就像十月怀胎，解决问题就像一朝分娩"

2. **反对两种错误**
   - **教条主义**：拿书本条文套实际情况，照猫画虎
   - **经验主义**：只凭过去经验，不研究新情况
   - 两者共同特点：思想懒惰，不愿深入实际

3. **调查的方法**
   - 开调查会：直接听当事人讲
   - "眼睛向下"：干部要向群众学习，向基层了解实情
   - 反对"老爷式的调查"——走马观花、敷衍塞责
   - 调查要有明确问题意识，不能漫无目的

4. **理论与实际的关系**
   - 马克思主义的"本本"是要学的，但必须同中国实际情况相结合
   - 任何理论都是特定时空下的产物，不能跨语境直接套用
   - 创造性地运用才是真正的掌握

**完整原文名句**：
- "没有调查，没有发言权"
- "中国革命斗争的胜利要靠中国同志了解中国情况"
- "马克思主义的'本本'是要学习的，但是必须同我国的实际情况相结合"
- "调查就是解决问题"

**现代应用**：
- 投研核心方法：一手信息 > 二手分析，访谈 > 研报，实地调研 > 数据库
- 反框架陷阱：任何分析框架（DCF/GARP/护城河）都是工具，不是答案本身
- 决策习惯：遇到判断分歧，先问"你调查过吗"，而不是"你的逻辑是什么"

## 二、军事战略类

### 论持久战（1938年）

**写作背景**：抗日战争初期，两种错误观点盛行——"速胜论"（轻敌）和"亡国论"（恐敌）。毛泽东系统驳斥两者，提出持久战理论。

**核心论断**：
- 中国不会速胜，也不会灭亡，抗战必然是持久战，最终胜利属于中国

**五对矛盾分析**：

| 比较维度 | 日本（强） | 中国（弱） |
|----------|-----------|-----------|
| 军事经济 | 强，但战争是退步的、小国的 | 弱，但正义的、大国的 |
| 国际环境 | 孤立、遭反对 | 同情，有援助可能 |
| 人力资源 | 少，难以补充 | 多，可动员 |
| 战争性质 | 侵略战争，失道寡助 | 反侵略，得道多助 |
| 持久能力 | 消耗战中逐渐劣势 | 持久战中逐渐转化优势 |

**三个战略阶段**：
1. **战略防御阶段**：敌强我弱，以空间换时间，保存有生力量
2. **战略相持阶段**：敌我僵持，游击战为主，积累力量，等待转机
3. **战略反攻阶段**：我强敌弱，全面反攻，夺取最终胜利

**战术原则**：
- 主动性：不打被动挨打的仗，争取主动权
- 灵活性：避免阵地战，发挥运动战、游击战优势
- 计划性：一切行动服从战略全局
- **"战略上藐视敌人，战术上重视敌人"**

**胜利条件**：
- 建立广泛的统一战线（团结一切可以团结的力量）
- 武装民众（人民战争）
- 改革政治（争取国内外同情与支持）

**完整原文名句**：
- "武器是战争的重要因素，但不是决定的因素，决定的因素是人不是物"
- "战争问题中的唯心论和机械论的倾向，是一切错误观点的认识论根源"
- "持久战的三个阶段"是中国抗战的战略路线

**现代应用**：
- 弱者战略：处于劣势时不硬刚，用时间换空间，保存核心资产
- 投资类比：熊市持仓策略——不预测底部，保留实力，等待拐点
- 竞争策略：对抗强大对手时，打不对称战（差异化定位、细分市场）
- 组织管理：重大变革分三阶段推进，切忌一蹴而就

### 十大军事原则（1947年）

**写作背景**：解放战争转入战略进攻阶段，毛泽东总结三大战役前的作战经验，提出具有普遍意义的作战原则。

**十大原则完整版**：

1. **先打分散孤立之敌，后打集中强大之敌**
   - 逻辑：积累小胜，提升实力，再打硬仗
   - 应用：先攻易目标积累信心和资源，再攻战略要地

2. **先取小城市中等城市和广大乡村，后取大城市**
   - 逻辑：农村包围城市——根据地建设先于城市攻坚
   - 应用：先占据细分市场，建立根据地，再进攻主流市场

3. **以歼灭敌军有生力量为主要目标，保守地方为次要目标**
   - 逻辑：消灭有生力量才是根本，占地盘是次要的
   - 应用：打击对手核心能力（人才、技术、客户），而非争抢市场份额数字

4. **每战集中绝对优势兵力，四面包围，力求全歼**
   - 逻辑：以十攻一，绝不打均势之战
   - 应用：集中资源在优势战场，不分兵作战

5. **不打无准备之仗，不打无把握之仗**
   - 逻辑：充分准备是胜利的前提，打有把握的仗
   - 应用：投资/决策前充分调研，不以"试试看"心态做重大决策

6. **发扬勇敢战斗、不怕牺牲、不怕疲劳和连续作战的作风**
   - 逻辑：精神意志是战斗力的组成部分
   - 应用：高强度工作期间保持战斗意志和纪律

7. **力求在运动中歼灭敌人（运动战为主）**
   - 逻辑：不死守阵地，在机动中寻找消灭敌人的机会
   - 应用：保持灵活性，不死守某一仓位/项目，在流动中寻找机会

8. **攻城：守备薄弱的坚决夺取，守备较强的等待时机**
   - 逻辑：区分可打和不可打的目标，量力而行
   - 应用：精准识别当前时机是否成熟，硬攻有准备之敌消耗过大

9. **以俘获敌人全部武器和大部人员来补充自己**
   - 逻辑：以战养战，消耗战反而壮大自己
   - 应用：竞争中吸纳对手的优秀人才、客户、渠道

10. **善于利用两个战役间的间隙，休息和整训部队**
    - 逻辑：持续战斗能力依赖有序的休整
    - 应用：高强度工作后必须有复盘和调整期，不能无脑连轴转

**核心逻辑**：集中优势 → 歼灭有生力量 → 以战养战 → 持续进攻

**现代应用综合**：
- 资源配置：集中资源在最有把握的战场，拒绝平均分散
- 竞争策略：打击对手核心能力，而非争夺表面指标
- 节奏管理：张弛有度，高强度进攻后必须有修整期

### 星星之火，可以燎原（1930年）

**写作背景**：1930年，红军处于艰难处境，党内悲观情绪蔓延，部分同志对革命前途失去信心。毛泽东写信驳斥悲观论，阐述革命前途必然光明的依据。

**核心观点**：

1. **批判两种错误情绪**
   - **悲观主义**：看不到革命力量的积累，只看到眼前困难
   - **盲目乐观**：对困难估计不足，脱离实际的浪漫主义
   - 正确态度：客观分析矛盾，在困难中看到发展趋势

2. **革命高潮的科学预判**
   - "它是站在海岸遥望海中已经看得见桅杆尖头了的一只航船"
   - "它是立于高山之巅远看东方已见光芒四射喷薄欲出的一轮朝日"
   - "它是躁动于母腹中的快要成熟了的一个婴儿"
   - 这三个比喻说明：革命高潮是可预见的，已有苗头，但还未全面到来

3. **农村包围城市的战略逻辑**
   - 中国是半殖民地半封建社会，农村是革命的广阔天地
   - 城市有敌人重兵，农村是薄弱环节
   - 根据地是革命的立足点，从农村积累力量再夺城市

4. **战略定力的哲学基础**
   - 信心来自对规律的认识，而不是盲目乐观
   - 等待是一种战略——不是消极等待，而是积极建设根据地
   - 长期主义：不以一时的困难否定长期趋势

**完整原文名句**：
- "星星之火，可以燎原"
- "它是站在海岸遥望海中已经看得见桅杆尖头了的一只航船"
- "我们党的任务不是等候革命，也不是鼓吹革命，而是积极地准备"

**现代应用**：
- 投资信念：在趋势早期保持信心，区分"苗头"和"全面爆发"，不急于求成
- 创业早期：星星之火阶段，核心任务是建立根据地（核心产品/核心客户），不急于全面扩张
- 个人成长：技能积累期不要被短期成果缺失打击，要看到"桅杆尖头"
- 战略耐心：长期主义不是坐等，是积极建设，等待拐点

### 游击战术（抗日游击战争的战略问题，1938年）

**写作背景**：抗战初期，针对轻视游击战的倾向，系统阐述游击战的战略地位和战术原则。

**核心观点**：

1. **游击战的战略地位**
   - 游击战不是战术问题，而是战略问题
   - 在战略防御和战略相持阶段，游击战是主要作战形式
   - 游击战的战略目的：牵制、消耗、瓦解敌人，保存和积累自己

2. **游击战六大原则**
   - **主动、灵活、计划地**执行攻势战斗
   - 配合正规战进行作战
   - 建立根据地（战略基地）
   - 防御中的进攻，持久中的速决，内线中的外线
   - 主动权的争夺（不打被动之仗）
   - 运动战：走、打、打了就走

3. **"十六字诀"**
   - **敌进我退，敌驻我扰，敌疲我打，敌退我追**
   - 核心：不与强敌正面硬拼，用运动消耗敌人

4. **根据地建设**
   - 根据地是游击战的后盾，没有根据地就没有持续作战能力
   - 在根据地内建立政权、军队、经济
   - 根据地要选在山地、森林等有利地形

**完整原文名句**：
- "敌进我退，敌驻我扰，敌疲我打，敌退我追"（十六字诀）
- "没有根据地，游击战争是不能长期存在和发展的"

**现代应用**：
- 弱者竞争策略：不正面硬拼，打游击战——差异化、细分市场、非对称竞争
- 创业策略：建立根据地（核心用户群/核心产品），不贸然全线进攻
- 个人策略：回避强敌优势领域，在对方薄弱处积累优势

## 三、政治智慧类

### 论联合政府（1945年）

**写作背景**：抗战即将胜利，国共两党争夺战后政治格局，毛泽东在七大作政治报告，阐述中共的政治主张。

**核心观点**：

1. **政治路线**
   - 废除国民党一党专政，建立民主联合政府
   - 放手发动群众，壮大人民力量
   - "我们共产党人好比种子，人民好比土地"

2. **三大作风（中共的精神特质）**
   - **理论和实践相结合**：反对本本主义，一切从实际出发
   - **密切联系群众**：把党的路线与群众的需求结合起来
   - **批评与自我批评**：党内民主，勇于自我纠错
   - 这三大作风使党区别于其他任何政党

3. **统一战线策略**
   - 团结一切可以团结的力量（包括民族资产阶级、开明士绅）
   - "求同存异"：在共同目标下包容分歧
   - 孤立最顽固的少数，争取最广泛的多数

4. **人民战争思想**
   - 兵民是胜利之本
   - 真正强大的力量在于人民，而不在于装备
   - "知无不言，言无不尽；言者无罪，闻者足戒"

**现代应用**：
- 组织领导：三大作风是高绩效团队的底层逻辑——知行合一、群众路线、自我批评
- 联盟策略：统一战线思想——在竞争中团结次要对手对抗主要对手
- 批评文化：建立制度化的批评和自我批评机制，防止组织僵化

### 关于正确处理人民内部矛盾的问题（1957年）

**写作背景**：社会主义建设初期，苏联斯大林问题暴露后，国内出现思想波动，毛泽东在最高国务会议上作报告，系统阐述处理矛盾的方法论。

**核心观点**：

1. **两类矛盾理论（最重要的理论贡献之一）**

   | 类型 | 特征 | 处理方法 |
   |------|------|----------|
   | 敌我矛盾 | 对抗性，根本利益冲突 | 专政方法（强制） |
   | 人民内部矛盾 | 非对抗性，基本利益一致 | 民主方法（说服教育） |

   - **关键判断**：正确区分两类矛盾，避免把人民内部矛盾当敌我矛盾处理（扩大化）
   - **最大危险**：将非对抗性矛盾激化为对抗性矛盾

2. **"团结——批评——团结"公式**
   - 从团结出发（肯定共同基础）
   - 经过批评（指出问题，说服教育）
   - 达到新的更高基础上的团结
   - 适用于一切人民内部矛盾的处理

3. **"百花齐放，百家争鸣"**
   - 文艺领域：百花齐放
   - 学术领域：百家争鸣
   - 核心精神：允许不同声音，在自由讨论中找到真理

4. **"长期共存，互相监督"**
   - 共产党与民主党派的关系原则
   - 多党合作是中国政治制度的特色
   - 监督本质上是一种互利关系

**完整原文名句**：
- "在我们面前有两类社会矛盾，这就是敌我之间的矛盾和人民内部的矛盾"
- "敌我之间的矛盾是对抗性的矛盾，人民内部的矛盾是非对抗性的"
- "百花齐放，百家争鸣"
- "团结——批评——团结"

**现代应用**：
- 冲突管理：处理团队矛盾前，先判断：这是原则性冲突（敌我）还是方法论分歧（内部）？
- HR管理：对可挽救的人用教育方法，对价值观根本冲突者用处置方法，切勿混淆
- 组织文化：建立安全的批评环境——"言者无罪，闻者足戒"

### 为人民服务（1944年）

**写作背景**：延安时期，张思德因烧炭牺牲，毛泽东在追悼会上发表演讲，阐述共产党人的人生观和价值观。

**核心观点**：

1. **生死观：重于泰山还是轻于鸿毛**
   - "为人民利益而死，就比泰山还重；替法西斯卖力，替剥削人民和压迫人民的人去死，就比鸿毛还轻"
   - 人的价值在于服务的对象和目的，而不在于地位高低

2. **批评与自我批评**
   - "只要我们为人民的利益坚持好的，为人民的利益改正错的，我们这个队伍就一定会兴旺起来"
   - 接受批评的前提：批评是正确的，为了人民利益
   - 批评不分地位高低——"不管是什么人，谁向我们指出都行"

3. **团结精神**
   - "我们都是来自五湖四海，为了一个共同的革命目标走到一起来了"
   - 大局意识高于个人得失
   - 互相关心、互相爱护、互相帮助

**完整原文名句**：
- "为人民服务"
- "为人民利益而死，就比泰山还重"
- "只要我们为人民的利益坚持好的，为人民的利益改正错的，我们这个队伍就一定会兴旺起来"

## 四、方法论综合

### 群众路线

**核心内容**：
- "从群众中来，到群众中去"
- 一切为了群众，一切依靠群众
- 把群众的意见集中起来，化为系统的意见，再回到群众中坚持下去

**完整循环**：
```
分散的群众意见 → 集中（科学化、系统化）→ 化为政策/路线 → 回到群众中 → 群众实践 → 再收集意见（循环往复）
```

**与"实践论"的关系**：
- 群众路线是实践论在群众工作中的具体应用
- 群众是实践的主体，党是认识和指导的组织
- 正确的路线必须通过群众检验

**现代应用**：
- 产品管理：用户调研 → 需求分析 → 产品设计 → 用户验证 → 迭代（群众路线的产品版）
- 管理决策：基层信息收集 → 管理层综合判断 → 决策落地 → 执行反馈 → 决策修正

---

### 实事求是

**核心内涵**：
- "实事"就是客观存在的一切事物
- "是"就是客观事物的内部联系，即规律性
- "求"就是研究
- 完整定义：一切从实际出发，理论联系实际，在实践中检验真理和发展真理

**三个层次**：
1. **认识层面**：不从主观愿望出发，而从客观实际出发
2. **方法层面**：通过调查研究了解实际，通过实践检验认识
3. **态度层面**：敢于承认错误，敢于修正路线

**与其他方法论的关系**：
- 实事求是是毛泽东思想的精髓
- 实践论提供认识论依据
- 矛盾论提供分析工具
- 调查研究是实践方法
